# chip-8 emulator
A simple chip-8 emulator

## Run
+ Follow [raylib](https://github.com/raysan5/raylib) build and installation guide for you respective operating system.
+ Clone this repo

```
$ cmake -S . -B build/
$ cmake --build build/
```

```
$ ./build/chip-8 /path/to-rom
```
You can use test roms inside `test-roms` directory
